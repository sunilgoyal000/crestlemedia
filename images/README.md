# Images Folder

Place your website images here.

## Required Images

### Logo
- `logo.png` or `logo.svg` - Company logo (recommended: 200x50px)

### Hero Section
- `hero-bg.jpg` - Hero background image (recommended: 1920x1080px)
- `hero-image.png` - Hero main image (recommended: 720x500px)

### Team/About
- `team-1.jpg` - Team member photo
- `team-2.jpg` - Team member photo
- `team-3.jpg` - Team member photo
- `about.jpg` - About section image (recommended: 600x400px)

### Services
- `service-social.jpg` - Social media marketing image
- `service-ads.jpg` - Paid advertising image
- `service-content.jpg` - Content creation image
- `service-seo.jpg` - SEO services image
- `service-email.jpg` - Email marketing image
- `service-brand.jpg` - Brand strategy image

### Portfolio
- `portfolio-1.jpg` to `portfolio-6.jpg` - Project images (recommended: 400x300px)

### Testimonials
- `client-1.jpg` - Client photo
- `client-2.jpg` - Client photo
- `client-3.jpg` - Client photo

### Favicon
- `favicon.ico` - Website favicon (recommended: 32x32px)

## Tips
- Use PNG for logos and images with transparency
- Use JPG for photographs
- Compress images for faster loading
- Use descriptive file names
